# 🧪 Exercícios de CSS – Nível Iniciante

> ⚠️ Instruções:
> - Utilize **CSS externo**
> - Crie um arquivo `index.html` e um `style.css`
> - Conecte o CSS ao HTML corretamente
> - Toos os exercicios deverão estar em um unico  arquivo HTML e um único arquivo CSS

---

## 1. Cor de fundo

Crie uma página com:
- Um título (`<h1>`)
- Um parágrafo (`<p>`)

Aplique:
- Uma cor de fundo para a página
- Uma cor diferente para o texto do título

---

## 2. Fonte e tamanho

- Altere a fonte do texto (ex: Arial)
- Defina tamanhos diferentes para:
  - Título
  - Parágrafo

---

## 3. Alinhamento  de texto

- Centralize um título na página
- Alinhe um parágrafo à direita

---

## 4. Cores em botões

Crie um botão (`<button>`) com:
- Cor de fundo
- Cor do texto
- Sem borda

---

## 5. 📦 Espaçamento (margin e padding)

Crie uma caixa (`<div>`) e adicione:
- Espaçamento interno (`padding`)
- Espaçamento externo (`margin`)

---

## 6. 🧱 Bordas

Crie uma caixa com:
- Borda sólida
- Cor personalizada
- Cantos arredondados (`border-radius`)

---

## 7. 🖱️ Hover (interação)

Crie um botão que:
- Mude de cor quando o mouse passar por cima (`:hover`)

---

## 🚀 Desafio Extra (Opcional)

- Combine vários conceitos em uma única página
- Estilize tudo para ficar visualmente agradável
